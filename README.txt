Мой учебный проъкт

Запуск сервера: node server


Механизмы создания и работы приложения:

1 Приложение реализовано на JavaScript без
использования клиентских JS-фреймвроков.

2 Получение списка накладных происходит через 
запросы к серверу на основе REST API.

3 В качестве сервера для работы REST API использовался JSON Server.

4 Данные, которые должны публиковаться сервером
при старте находятся в файле db.json.

5 Все запросовы к серверу происходят по принципу AJAX.

6 Дополнительные библиотеки(Jest и Json-server) установлены
при помощи пакетного менеджера npm через командную строку.



Функционал:

1 На главной странице приложения отображается список счетов (накладных).

2 Создание и редактирование накладных происходит через форму(<form>).
    
3 В каждой строке списка на главной странице присутствуют кнопки 
«Изменить» и «Удалить» для выполнения соответствующих действий
над конкретной накладной при помощи созданной формы.
    
4 Для списка на главной странице приложения работает 
фильтрация отображаемых строк по значению выбранного поля 
(с использованием операции “равно”).
    
5 Для списка на главной странице приложения работает
полнотекстовый поиск по всем полям.
    
6 Для списка на главной странице приложения работает 
сортировка по одному полю в трех режимах: “без сортировки”, 
“по возрастанию”, “по убыванию”.
    
7 Фильтрация, сортировка и поиск для списка на главной странице приложения
работают совместно, т.е. в запросе учитывается всё, что выбрал пользователь.

8 Для приложении есть светлая и тёмная темы с переключателем
и запоминаним выбранной(с помощю localstorage).

9 Приложение адаптировано под экраны разного размера.
